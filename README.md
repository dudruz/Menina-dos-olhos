# 👁️ Menina dos Olhos — Óptica Premium

> Conforto, beleza e qualidade para o seu olhar.

Um site institucional moderno, responsivo e altamente focado em conversão para uma marca óptica.
Desenvolvido com foco em **experiência premium**, **design estratégico** e **performance visual**.

---

## 🧠 Sobre o Projeto

O projeto **Menina dos Olhos** foi desenvolvido para transformar a presença digital de uma ótica em uma **marca forte, moderna e confiável no digital**.

Mais do que um site, ele funciona como uma:

✔️ vitrine digital
✔️ ferramenta de conversão
✔️ ponto de autoridade da marca

---

## 🎯 Objetivo

Este site foi pensado para:

* gerar mais contatos via WhatsApp
* aumentar a percepção de valor da marca
* facilitar a escolha de produtos
* reduzir dúvidas do cliente
* transformar visitantes em leads

---

## 🔥 Diferenciais do Projeto

* 🎨 Design premium inspirado em Apple/Stripe
* ⚡ Interface rápida e leve
* 📱 Totalmente responsivo (mobile-first)
* 🧠 UX focado em conversão
* 💬 Integração direta com WhatsApp
* ✨ Animações suaves e modernas
* 🧩 Estrutura escalável para futuras integrações
